# Weekday Dashboard

This repo contains the source code of Weekday dashboard assignment.

## Installation

Clone the repo

```bash
git clone https://github.com/SanjayM-2002/Weekday-dashboard.git
```
```bash
cd Weekday-dashboard
```
```bash
npm i
```
```bash
npm run dev
```
